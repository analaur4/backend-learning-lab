# História 01 — Criar coleção de testes no Postman ou Insomnia

**Como** pessoa desenvolvedora iniciante,  
**quero** testar manualmente os endpoints da API,  
**para** validar o funcionamento do CRUD.

## Descrição

Criar uma coleção no Postman, Insomnia ou ferramenta similar com todos os endpoints da API.

## Endpoints mínimos

- CRUD de Cliente
- CRUD de Produto
- CRUD de Pedido

## Critérios de aceite

- Deve existir uma coleção com os endpoints organizados.
- Deve ser possível criar, listar, buscar, atualizar e deletar clientes.
- Deve ser possível criar, listar, buscar, atualizar e deletar produtos.
- Deve ser possível criar, listar, buscar, atualizar e deletar pedidos.
