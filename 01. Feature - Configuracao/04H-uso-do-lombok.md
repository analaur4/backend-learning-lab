# História 04 — Configurar Lombok no projeto

**Como** pessoa desenvolvedora iniciante,  
**quero** utilizar Lombok no projeto,  
**para** reduzir código repetitivo nas classes Java.

## Descrição

Adicionar e utilizar Lombok nas entidades e DTOs do projeto.

## Explicação breve

O **Lombok** é uma biblioteca que gera automaticamente códigos comuns, como getters, setters, construtores, `equals`, `hashCode` e `toString`, usando anotações.

Exemplos de anotações comuns:

- `@Getter`
- `@Setter`
- `@NoArgsConstructor`
- `@AllArgsConstructor`
- `@Builder`

## Critérios de aceite

- O Lombok deve estar configurado no projeto.
- As classes podem usar anotações do Lombok.
- O projeto deve compilar corretamente com Lombok.
- A IDE deve reconhecer as anotações do Lombok.
