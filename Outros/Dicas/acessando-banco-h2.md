# Acessando banco de dados H2 Local

Conforme foi configurado que o path do banco de dados é `/h2-console` em: `spring.h2.console.path`, você pode acessar o banco pelo navegador usando o caminho: `localhost:8080/h2-console` e adicionar as informações necessárias de acesso como URL, usuário e senha (caso tenha configurado).

Será aberto uma tela com um console para escrita de queries.
