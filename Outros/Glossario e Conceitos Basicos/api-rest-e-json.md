# Conceitos sobre API REST e JSON

## O que é uma API REST?

REST é um padrão de comunicação entre sistemas utilizando HTTP.

---

### Exemplos de endpoints

```http id="yy3hr3"
GET /clientes
POST /clientes
PUT /clientes/1
DELETE /clientes/1
```

---

### Métodos HTTP

| Método | Objetivo     |
| ------ | ------------ |
| GET    | Buscar dados |
| POST   | Criar        |
| PUT    | Atualizar    |
| DELETE | Remover      |

---

## O que é JSON?

JSON é um formato utilizado para troca de informações entre sistemas.

---

### Exemplo JSON

```json id="y31rte"
{
  "nome": "Ana",
  "email": "ana@email.com"
}
```
