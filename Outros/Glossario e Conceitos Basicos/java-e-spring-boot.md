# Conceitos sobre Java e Spring Boot

## O que é Java?

Java é uma linguagem de programação orientada a objetos criada pela Sun Microsystems (atualmente pertencente à Oracle).

Ela é amplamente utilizada no mercado para:

* APIs backend
* Sistemas bancários
* Microsserviços
* Aplicações corporativas
* Android (legado)

### Principais características

* Orientada a objetos
* Fortemente tipada
* Multiplataforma (“Write Once, Run Anywhere”)
* Grande ecossistema de bibliotecas
* Muito utilizada em empresas

---

## O que é Spring Boot?

Spring Boot é um framework criado para facilitar o desenvolvimento de aplicações Java.

Ele reduz configurações complexas e acelera a criação de APIs e sistemas backend.

### O que o Spring Boot oferece?

* Criação rápida de APIs REST
* Injeção de dependência
* Integração com banco de dados
* Servidor embutido (Tomcat)
* Tratamento de erros
* Segurança
* Validações
* Monitoramento

### Objetivo principal

Facilitar o desenvolvimento de aplicações Java modernas.
